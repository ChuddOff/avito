## Avito AI Assistant

Тестовое задание на frontend-стажировку Avito (весенняя волна 2026).

Приложение представляет собой личный кабинет продавца, в котором можно:

- просматривать список объявлений;
- открывать карточку объявления;
- редактировать объявление;
- получать AI-помощь при редактировании:
  - улучшение / генерация описания;
  - рекомендация цены.

---

## Что реализовано

### Основной сценарий

- `/` — редирект на `/ads`
- `/ads` — список объявлений
- `/ads/:id` — просмотр объявления
- `/ads/:id/edit` — редактирование объявления

### На странице списка

- загрузка объявлений с backend API;
- поиск по названию;
- фильтрация по категориям;
- фильтр `Только требующие доработки`;
- сортировка;
- пагинация;
- синхронизация состояния списка с URL;
- loading / error / empty states.

### На странице просмотра

- отображение основной информации об объявлении;
- отображение характеристик в зависимости от категории;
- блок `Требуются доработки` со списком незаполненных полей;
- переход к редактированию;
- возврат к списку.

### На странице редактирования

- форма на React Hook Form + Zod;
- динамические поля в зависимости от категории;
- валидация обязательных полей;
- сохранение изменений через `PUT /items/:id`;
- AI-кнопки:
  - `Придумать / Улучшить описание`
  - `Узнать рыночную цену`
- AI-ответ показывается отдельно и применяется в форму только по явному действию пользователя.

## Архитектура

Проект организован в стиле **FSD (Feature-Sliced Design)**, но без избыточного дробления.

### Основные слои

- `app` — инициализация приложения, роутинг, провайдеры
- `pages` — страницы
- `widgets` — крупные блоки страницы
- `features` — пользовательские сценарии
- `entities` — доменные сущности
- `shared` — переиспользуемые утилиты, API-клиенты, UI-примитивы

### Принятые правила

- shadcn-компоненты лежат в `shared/ui`;
- внешние импорты между слайсами — только через public API (`index.ts`);
- бизнес-компоненты не выносятся в `shared/ui`;
- логика AI вынесена в отдельные feature-слайсы;
- форма редактирования не знает деталей AI-клиента напрямую.
